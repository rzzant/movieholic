# 🎬 MovieHolic

## 🚀 Overview

MovieHolic is a web application that allows users to explore, search, and discover movies. It provides details like ratings, genres, release dates, and descriptions in a clean and user-friendly interface.

The goal of this project is to create a smooth movie browsing experience similar to platforms like IMDb or Netflix (basic level).

---

## 🎯 Features

* 🔍 Search for movies by name
* 🎥 View movie details (rating, overview, release date)
* 📂 Browse trending / popular movies
* ❤️ Add movies to favorites (if implemented)
* 📱 Responsive UI for all devices

---

## 🛠️ Tech Stack

* **Frontend:** React.js / HTML / CSS / JavaScript
* **API:** TMDB API (The Movie Database)
* **Other Tools:** Axios / Fetch API

## 🔑 API Setup

1. Go to https://www.themoviedb.org/
2. Create an account and generate an API key
3. Add your API key in the project:

```
const API_KEY = "your_api_key_here";
```
