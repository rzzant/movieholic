# Movieholic

## Overview

Movieholic is a web application that allows users to explore movies, view details, and discover trending and popular content using a movie database API.

## Problem

Users often struggle to find relevant movie information across multiple platforms.

## Solution

Movieholic provides a centralized interface to:

* Browse trending and popular movies
* Search for specific movies
* View detailed information such as ratings, release date, and overview

## Features

* Movie search functionality
* Trending and popular movies listing
* Detailed movie information
* Responsive user interface
